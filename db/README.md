# dbフォルダ

データベース関連のDDL、ストアドプロシージャ、ビュー、マイグレーションスクリプトを格納します。

## ディレクトリ構造

```
db/
├── tables/             # テーブル定義（CREATE TABLE文）
├── stored-procedures/  # ストアドプロシージャ
├── views/              # ビュー定義
└── migrations/         # マイグレーションスクリプト
```

## 命名規則

- **テーブル定義**: `[スキーマ名].[テーブル名].sql`（例: `dbo.users.sql`）
- **ストアド**: `[スキーマ名].[プロシージャ名].sql`（例: `dbo.usp_get_users.sql`）
- **ビュー**: `[スキーマ名].[ビュー名].sql`（例: `dbo.vw_active_users.sql`）
- **マイグレーション**: `[YYYYMMDD]_[連番]_[説明].sql`（例: `20260322_001_add_users_table.sql`）

## ルール

- 各SQLファイルは自己完結型とし、必要な依存関係をコメントで記載する
- テーブル定義にはカラムのコメント（COMMENT ON または説明コメント）を含める
- ストアドプロシージャにはパラメータと戻り値の説明をヘッダーコメントに記載する
- SQLレビュー時は `sql-review` スキルを参照する
